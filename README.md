# Truco Gaudério — Testes Automatizados com Pytest

Este repositório contém o trabalho desenvolvido para a disciplina **Qualidade de Software** sobre Testes de Software Automatizados, utilizando o **Pytest** para validar o comportamento e as regras do jogo **Truco Gaudério**.

---

## Objetivo

O objetivo deste projeto é **projetar, implementar e executar testes automatizados** para garantir que as funcionalidades do jogo Truco estejam corretas, cobrindo regras, rodadas, pontuação, envidos e demais comportamentos esperados no jogo gaúcho.
